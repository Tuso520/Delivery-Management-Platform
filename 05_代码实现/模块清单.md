# 模块清单

## 前端页面模块

- `dashboard/`：工作台、数据看板、待办。
- `project/`：项目台账、项目档案、文件审核、日报、复盘。
- `workflow/`：交付流程。
- `checklist/`：检查模板。
- `archive/`：档案模板和档案项。
- `template/`：文档模板。
- `knowledge/`：知识库。
- `tools/`：工具中心。
- `performance/`：目标绩效、技能评估。
- `organization/`：组织架构、用户、角色权限。
- `system/`：系统配置、日志、审批、备份、集成。

## 后端业务模块

- AuthModule、UserModule、RoleModule、PermissionModule。
- ProjectModule、ProjectMemberModule、ProjectPaymentModule、ProcessModule。
- ArchiveTemplateModule、ProjectArchiveModule、FileModule、AttachmentModule、ReviewModule。
- ChecklistModule、WorkflowModule、TemplateModule、KnowledgeModule、ToolModule。
- ReportModule、OkrModule、PlatformModule、NotificationModule。
- CountryModule、CurrencyModule、LanguageModule、OperationLogModule、SystemConfigModule、DashboardModule。

## 关键脚本

- `scripts/local-test-server.mjs`：本地模拟服务。
- `scripts/local-docker.ps1`：Windows 本地 Docker 启停。
- `deploy-git.sh`：服务器 Git 拉取部署。
- `scripts/package-release.sh`、`scripts/package-release.ps1`：发布包辅助脚本，保留用于兼容旧流程。
