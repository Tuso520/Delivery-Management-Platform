# 项目资料上传说明

## 适用范围

本说明用于项目经理、实施工程师、软件工程师在项目档案和知识库中上传过程资料时参考。

## 上传要求

1. 文件名称应包含项目简称、流程节点、资料名称和版本号。
2. 合同、验收、结算等关键资料优先上传 PDF 扫描件。
3. 点表、设备清单、成本预算等结构化资料优先上传 Excel。
4. 会议纪要、岗位职责、操作说明优先上传 Word 或 Markdown。
5. 项目汇报、复盘说明可上传 PPT。

## 审批说明

涉及合同、成本、签证、验收和复盘的资料，上传后应进入审批流程。审批通过后，平台展示最新有效版本；审批驳回时，上传人应根据意见重新提交。
