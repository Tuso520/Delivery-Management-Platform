# 09 部署运维

本目录记录 Docker 本地测试、Git 推送部署、数据保护、备份和回滚。

## 文件清单

- `部署运维手册.md`：日常开发、测试、部署、回滚操作。
- `Git推送部署与数据保护方案.md`：GitHub 到服务器的部署转换方案。
- `Docker本地测试部署.md`：本地 Docker 测试说明。
- `2026-07-03_Docker本地部署执行记录.md`：历史 Docker 验收记录。

## 部署原则

- 本地验证先于 Git 推送。
- GitHub 是服务器部署的唯一代码来源。
- 生产数据通过 volume 和备份保护，不随代码更新覆盖。
