# 发布验收清单

## 发布包

- [ ] SHA256 校验通过
- [ ] 压缩包根目录直接包含 `docker-compose.yml`
- [ ] 不包含 `.env`、node_modules、dist、测试、旧压缩包和历史文档
- [ ] `RELEASE_ID` 与文件名一致

## 基础服务

- [ ] MySQL、Redis、MinIO 健康
- [ ] 后端健康接口返回成功
- [ ] 前端健康接口返回成功
- [ ] `/build-info.json` 与 `RELEASE_ID` 一致

## 代码质量

- [ ] 后端 `pnpm type-check` 通过
- [ ] 后端 `pnpm test -- --runInBand` 通过
- [ ] 后端 `pnpm build` 通过
- [ ] 前端 `pnpm type-check` 通过
- [ ] 前端 `pnpm test` 通过
- [ ] 前端 `pnpm build` 通过
- [ ] Prisma schema 校验通过
- [ ] 生产迁移使用 `prisma migrate deploy`，没有 `db push --accept-data-loss`

## 界面

- [ ] 登录页桌面和移动端布局正常
- [ ] 侧栏完整、可折叠、可滚动，移动端抽屉正常
- [ ] 顶栏标题、语言和用户菜单正常
- [ ] 所有二级菜单页面没有内容遮挡或异常横向空白
- [ ] 表格、筛选、表单、弹窗、分页和空状态样式一致

## 业务

- [ ] 数据看板项目和回款数据正常
- [ ] 项目查看、编辑、成员和回款可用
- [ ] 项目档案显示完整多级目录
- [ ] 项目过程记录支持文件、照片和视频
- [ ] 日报、周报、月报及工时统计可用
- [ ] 流程、检查模板、档案模板、文档模板有真实内容
- [ ] 知识附件、培训资料和录屏可上传下载
- [ ] OKR/KPI 创建和评分流程可用
- [ ] 国家、汇率、语言翻译和系统设置可用

## 权限与审计

- [ ] 超级管理员显示全部菜单
- [ ] 普通角色只显示授权菜单
- [ ] 角色权限矩阵可保存
- [ ] 项目数据范围限制有效
- [ ] 写操作和敏感下载产生操作日志

## 回滚

- [ ] 发布前数据库和 MinIO 备份存在
- [ ] 旧源码目录和 rollback 镜像存在
- [ ] `bash deploy.sh rollback` 可执行
