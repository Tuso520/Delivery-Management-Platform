# 平台优化实施计划

1. 扩展技能评估和评分审批数据结构，生成 Prisma 迁移。
2. 修复看板、项目台账、档案层级与统计，合并项目过程记录入口。
3. 补齐日报、周报、月报和八项工时统计。
4. 重构知识分类和连续浏览，完善文件查看入口。
5. 重构目标绩效配置顺序、分类、评分标准和逐级审批。
6. 建立技能库和人员技能评分。
7. 扩展系统配置、公开登录配置和飞书集成。
8. 统一 Arco 风格颜色、表格、选择器和工具栏。
9. 执行 Prisma 校验、前后端类型检查、测试、Lint 和生产构建。
10. 使用发布白名单生成新的 tar.gz、SHA256 和 manifest，不启动本地服务。
