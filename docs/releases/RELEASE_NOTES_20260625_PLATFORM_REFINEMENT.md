# 2026-06-25 平台优化发布说明

## 主要调整

- 看板改为回款优先，移除待办卡片并调整项目列表顺序。
- 项目台账隐藏项目编号，增加人民币折算和项目负责人。
- 项目档案按父子目录展示，项目过程记录并入档案。
- 日报、周报、月报补齐示例数据，工时统计扩展为八项。
- 知识库改为一级分类连续浏览，并完善附件预览下载。
- 目标与绩效支持四类目标、指标权重、评分标准和顺序审批。
- 技能库扩展为项目、电气、软件、调试、安全、质量和通用能力。
- 系统参数扩展，登录文案可配置，接口集成增加飞书。
- 全局视觉切换为 Arco 风格浅色扁平界面。

## 数据兼容

- Prisma 保持 `5.22.0`。
- `skill_assessments` 新增自评、上级评分和最终成绩字段。
- 种子脚本只创建缺失数据，不覆盖服务器已有业务内容或管理员配置。
