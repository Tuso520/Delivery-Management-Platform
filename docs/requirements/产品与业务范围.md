# 产品与业务范围

## 产品定位

交付管理平台是面向跨国家交付中心的企业内部业务系统。平台围绕项目交付全过程组织人员、回款、档案、过程记录、报告、流程标准、知识和绩效数据，不作为通用网盘或独立财务系统。

## 使用对象

- 交付负责人、国家负责人和项目经理
- 电气、软件、采购、财务、HSE 和标准化人员
- 系统管理员、审计人员和受限外部协作人员

## 当前菜单

### 工作台

- 数据看板：项目状态、阶段、风险、合同和回款统计
- 我的待办：审批、资料上传、审核和退回事项

### 项目管理

- 项目台账：项目基础信息、负责人、阶段、风险和金额
- 项目档案：按档案模板生成的多级目录、文件和审核状态
- 项目过程记录：会议、问题、进度、照片、视频和相关文件
- 文件审核：待审核资料集中处理
- 工时与周报：日报、周报、月报及项目工时汇总
- 项目复盘：问题、经验、责任人、整改和关闭验证

### 标准与知识

- 交付流程：分类化流程文档和版本内容
- 检查模板：可复用检查标准，不承担项目现场记录职责
- 档案模板：项目档案目录、责任角色和审核规则
- 文档模板：标准文档版本、发布和下载
- 知识库：Markdown 正文及文档、表格、PDF、演示和图片附件
- 工具中心：交付相关工具入口
- 培训记录：培训人、参与人、资料、录屏和完成情况

### 绩效与团队

- 目标与绩效：OKR/KPI、关键结果、评分流程、评分人和计算规则
- 技能评估：技能定义、等级和季度评估

### 组织与权限

- 组织架构、用户管理、角色权限
- 权限按模块和查看、下载、上传、操作动作分配

### 系统设置

- 国家、币种汇率、语言翻译
- 通知、审批、操作日志、系统参数
- 存储备份和接口集成

## 业务边界

- 保留项目合同金额和回款节点，不建设独立商务审批或完整财务核算。
- 不建设客户、供应商、采购、物流、费用和资产独立台账。
- 不建设人员证件模块。
- 国家、币种、语言、项目类型、阶段、角色和模板均从配置或字典读取。
- 文件存储在私有 MinIO，数据库只保存索引和业务关联。
