# 安全性能方案

## 鉴权与权限

- 使用 JWT Bearer Token。
- 后端 Guard 做登录、角色和权限码校验。
- 前端权限只控制显示，不能替代后端校验。
- 密码使用 bcrypt 哈希存储。

## 文件安全

- 文件存储在 MinIO，数据库只保存索引。
- 上传限制文件类型、大小和业务关联。
- 下载、预览、审核均需鉴权。
- 预览失败不得暴露对象存储内部路径。

## 日志审计

- 权限变更、文件下载、备份下载、合同和成本查看必须写入 OperationLog。
- 操作日志保留用户、模块、动作、目标、IP、结果和时间。

## 数据保护

- 服务器部署前先备份数据库和 MinIO 数据。
- 部署脚本不得覆盖生产 `.env`。
- 不得删除 Docker volume。
- 迁移失败必须回滚代码和镜像。

## 性能

- 列表接口分页，禁止无上限返回。
- 常用筛选字段建立索引。
- 大文件走对象存储直传或流式处理。
- 前端列表页避免一次渲染过大数据量。
