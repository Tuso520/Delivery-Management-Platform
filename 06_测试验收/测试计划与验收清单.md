# 测试计划与验收清单

## 自动化检查

| 检查项 | 命令 |
| --- | --- |
| 前端类型检查 | `pnpm --dir delivery-platform-web type-check` |
| 前端测试 | `pnpm --dir delivery-platform-web test` |
| 前端构建 | `pnpm --dir delivery-platform-web build` |
| 后端类型检查 | `pnpm --dir delivery-platform-server type-check` |
| 后端测试 | `pnpm --dir delivery-platform-server test` |
| 后端构建 | `pnpm --dir delivery-platform-server build` |
| Docker 配置 | `docker compose --env-file .env.example -f docker-compose.yml config -q` |
| 生产 Compose 配置 | `docker compose --env-file .env.example -f docker-compose.yml -f docker-compose.prod.yml config -q` |

## 页面验收

- 登录页：账号密码可登录，错误密码有提示。
- 主布局：菜单、面包屑、用户区正常显示。
- 项目台账：列表、筛选、分页、详情入口正常。
- 知识库：分类不重复，文章列表、附件上传、在线预览正常。
- 系统设置：选择框、按钮、表格列名无乱码。

## 知识库预览专项

| 类型 | 验收方式 |
| --- | --- |
| doc/docx | 打开文档型预览，显示文件名、版本、摘要或页数 |
| xls/xlsx | 打开表格型预览，显示 Sheet 和示例数据 |
| ppt/pptx | 打开幻灯片型预览，显示页数和缩略信息 |
| pdf | 打开 PDF 阅读区 |
| png/jpg | 打开图片预览区 |

## 验收标准

全部检查命令退出码为 0，本地模拟服务关键接口返回 200，页面核心文案无乱码，GitHub 推送成功。
