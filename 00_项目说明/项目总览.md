# 项目总览

## 项目定位

交付管理平台是面向软件公司交付中心的企业级管理系统，用于统一管理项目交付过程、项目档案、文件审核、交付标准、知识库、团队绩效和组织权限。

## 目标用户

- 交付总监：查看项目整体进度、风险、绩效和资源情况。
- 项目经理：维护项目台账、交付阶段、档案和项目复盘。
- 实施顾问：上传交付文件、执行检查清单、提交日报。
- 审核人员：审核项目文件、档案材料和关键节点。
- 知识管理员：维护知识分类、文章、版本和附件预览。
- 系统管理员：维护组织、用户、角色、权限、国家、币种、语言、审批和集成配置。

## 系统范围

当前源码覆盖以下业务域：

- 工作台：数据看板、我的待办。
- 项目管理：项目台账、项目档案、文件审核、工时与日报、项目复盘。
- 标准与知识：交付流程、检查模板、档案模板、文档模板、知识库、工具中心、培训记录。
- 绩效与团队：目标与绩效、技能评估。
- 组织与权限：组织架构、用户管理、角色权限。
- 系统设置：国家配置、币种汇率、语言翻译、通知规则、审批配置、操作日志、系统参数、存储备份、接口集成。

## 工程状态

- 前端：Vue 3 + TypeScript + Arco Design Vue，路由和页面已按业务域组织。
- 后端：NestJS 11 + Prisma 5，API 前缀为 `/api/v1`，Swagger 路径为 `/api/docs`。
- 数据：MySQL、Redis、MinIO 通过 Docker Compose 编排。
- 本地模拟：`scripts/local-test-server.mjs` 提供登录、菜单、知识库预览、文件等模拟接口。
- 部署：GitHub 仓库为 `Tuso520/Delivery-Management-Platform`，服务器部署采用 Git 拉取和 Docker Compose。
